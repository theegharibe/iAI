# iAI project
## 1. Secure File Uploading & Analysis

Supports multiple file types (text, PDF, data files, logs, and more).

Extracts and processes information entirely offline — no internet communication.

## 2. Personal Knowledge Base

All extracted information is added to the AI’s internal knowledge structure.

The knowledge becomes permanently available for future queries.

## 3. Intelligent Interactions

Ask direct questions about your files, such as:

“Give me a summary”

“Provide a security analysis”

“Show me the trends”

## Advanced capabilities include:

Generating summaries

Predictive modeling

Security assessments

Log and data analysis

Main Advantages

Complete anonymity and privacy — everything is processed locally.

High security — no external servers or cloud involvement.

Requirements

## All you need is:

Ollama installed

At least one local model such as:

- phi3

- llama3:8b

## How to Start

Download iAI.exe from the Releases section.

Run the file — no installation needed.

Upload your files → ask your questions → get instant analysis.
